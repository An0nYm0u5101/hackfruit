---
title: Cat
tags: Linux PrivEsc
---

Reads contents from files.

```sh
# Cron jobs
cat /etc/crontab
# Messages before login
cat /etc/issue
# Messages after login
cat /etc/motd
# Users
cat /etc/passwd
# Nameserver
cat /etc/resolv.conf
# 
cat /etc/shadow
#
cat /etc/sudoers
# Display line numbers
cat -n passwords.txt
```
